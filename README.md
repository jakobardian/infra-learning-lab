# Infrastructure Learning Lab
Repository ini berisi dokumentasi praktik dan setup infrastructure lab
sebagai bagian dari learning journey saya di bidang IT.

Tujuan utama repository ini adalah membangun pemahaman dan skill fundamental teknis untuk
jalur **IT Support - SysAdmin - DevOps** melalui praktik secara langsung.

---

## Objectives 
- Membangun skill sistem operasi, jaringan, dan server
- Mengembangkan mindset operasional serta kemampuan troubleshooting
- Mendokumentasikan praktik infrastructure secara reproducible sebagai portofolio teknis

---

## Lab-Scope
- Linux
- Networking
- SQL
- Docker
- Cloud 

Setiap topik disertai dokumentasi setup, konfigurasi dasar,
serta catatan troubleshooting jika diperlukan.

---

## Documentation Approach
- Seluruh dokumentasi ditulis berdasarkan praktik langsung
- Setiap keputusan teknis dicatat secara singkat dan jelas

---

# Notes
Repository ini bersifat **living documentation** dan akan diperbarui
secara bertahap sejalan dengan progres pembelajaran.
