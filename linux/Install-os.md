# Installation Operating System Linux 

## Objective
Melakukan instalasi OS Linux Ubuntu Server di virtual machine sebagai environment lab
untuk pembelajaran Linux.

## Prerequisite
- Virtual Machine: VMware Workstation Pro (version: 25H2)
- Operating System: ISO Ubuntu Server LTS (version: 24.04.3)

## Installation Steps

### 1. Download Linux Ubuntu ISO
ISO diunduh dari website resmi Ubuntu:
https://ubuntu.com/download/server

### 2. Installation Linux Ubuntu OS 
- Set virtual machine configuration: (*custom/advanced)
- Set virtual machine hardware compatibility (*Workstation 25H2)
- Set virtual machine guest OS installation: (*install from: Ubuntu ISO)
- Set virtual machine name & location
- Set virtual machine processor configuration (*2)
- Set virtual machine memory (*7GB)
- Set virtual machine network type (*NAT)
- Set virtual machine I/O controller type (*LSI Logic)
- Set virtual machine disk type (*SCSI)
- Set virtual machine disk (*Create a new virtual disk)
- Set virtual machine disk capacity (*30GB)
- Set up Ubuntu server personalization
- Finish
