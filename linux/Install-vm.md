# Virtual Machine Installation

Dokumentasi ini berisi langkah instalasi virtual machine VMware Workstation Pro 25H2 version.

---

## Objectives
Menyiapkan aplikasi virtual machine VMware Workstation yang digunakan sebagai learning lab virtual berbasis
OS Linux untuk praktik administrasi sistem linux.

---

## Environment
- Host OS : Windows

---

## Prerequisite
- Software: VMware Workstation Pro (version: 25H2)

---

## Installation Step

### 1. Download VMware Workstation
Download virtual machine VMware Workstation pada website resmi vmware.com
source: https://www.vmware.com/products/desktop-hypervisor/workstation-and-fusion

### 2. Installation Process
- Jalankan file installer
- Accept End-User License Agreement
- Ikuti proses instalasi hingga selesai
- Finish
